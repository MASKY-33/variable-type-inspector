# Variable Type Inspector
This program demonstrates how Python handles different data types.  
It creates three variables — an integer, a string, and a boolean — and prints their data types using the built‑in `type()` function.

## Features
- Contains three predefined variables:
  - an integer  
  - a text string  
  - a boolean value  
- Uses the `type()` function to inspect each variable  
- Prints the data type of every variable to the console  

# How to Use
Simply run the program.  
No user input is required.  
The program will immediately display the data types of the three variables.

# Learning Purpose
This small script was created to practice essential Python fundamentals:

- understanding basic data types  
- using the `type()` function  
- printing values to the console  
- working with integers, strings, and booleans  

It serves as a simple introduction to how Python represents and identifies different kinds of data.
